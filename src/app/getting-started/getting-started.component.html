How does FastEstimator work?

In FastEstimator, users only need to interact with three main APIs for any deep learning task. Each one represents one part of the machine learning process.

Pipeline

Pipeline takes care of loading and preprocessing data

Network

Network compiles all trainable and differentiable model operations and the loss.

Estimator

Finally, Estimator manages the training loop.
