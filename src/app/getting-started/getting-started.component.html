The flexible high-level open-source machine learning framework that will allow you to go wherever you want in the AI world.

Why FastEstimator?

FastEstimator provides all the simplicity and flexibility needed to easily build a high-performance deep learning model and run it anywhere.

Simple...

Getting started is very easy as FastEstimator is built upon Tensorflow2 and structured around only 3 main APIs. With FastEstimator, you can implement and train high performance models in only a few lines of code!

Yet flexible!

FastEstimator provides advanced functionalities and enables complex training schemes such as multi-model training and progressive training. Users can customize any part of the model development.

Useful AI modules

FastEstimator offers functions to facilitate the prototyping and production process. For example, the model interpretation module contains visualization tools such as feature UMAP to help users build more robust models.

Easy to scale

Distributed training usually requires non-trivial effort. In FastEstimator, all modules are designed to be distribution-friendly: users can scale their training and evaluation across multiple devices without any change of code.

How does FastEstimator work?

In FastEstimator, users only need to interact with three main APIs for any deep learning task. Each one represents one part of the machine learning process.

Pipeline

Pipeline takes care of loading and preprocessing data

Network

Network compiles all trainable and differentiable model operations and the loss.

Estimator

Finally, Estimator manages the training loop.
